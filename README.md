# Company Website
A modern and responsive software company website built with React, Vite, and Tailwind CSS. The website showcases company services, products, industry solutions, case studies, and contact information with a clean and user-friendly interface.

## Features
- Modern and responsive design
- Service and product showcas
- Industry solutions section
- Case studies and portfolio
- Contact and inquiry forms
- Optimized for desktop, tablet, and mobile devices

## Tech Stack
- React
- Vite
- Tailwind CSS
- TypeScript
